Overview:

What the Counter class does

The counter component is a resuable JavaScript components that displays numeric value and allows users to work with three interactive buttons like Increment, Decrement and Reset.

It is designed using Object-Oriented Programming principles, where it includes features like the count never goes below zero and when the value reaches zero, the Decrement and Reset buttons fades out which visually indicates that they are inactive.

What problem does it solves

The Counter solves the problem of repeatedly creating similar counters without repeating code. Its subclass, StepCounter,extends the functionality by allowing the numeric value to increase or decrease showcasing inheritance and polymorphism.

How to use it:

To get started, you need to create a container with an ID in your HTML where the counter will be displayed.

Example HTML container

<div id="counter-container"></div>

Each counter should have its own ID since the JavaScript uses those ID's to connect with specific containers. When the JavaScript runs, the counter component starts displaying numeric value along with controlling buttons inside that container.

Example JavaScript instantiation

import { Counter, StepCounter } from "./modules/counter.js";

const container1 = new Counter("#counter-container1");
const container2 = new Counter("#counter-container2", 4);
const newCounter = new Counter("#counterCon");
const stepCounter = new StepCounter("#step-counter", 0, 5);

This example imports the Counter and StepCounter classes from counter.js and creates counters. Two counters increases at initial value 0, one of them increases from 4, meanwhile the step counter increases and decreases by 5, demonstrating inheritance and polymorphism.
