What is Counter?

A counter component is a resuable piece of JavaScript that displays numeric value and allows users to work with three interactive buttons like increment, decrement and reset.

It is designed using Object-Oriented Programming principles, where it includes features like the count never goes below zero and when the value is zero, the Decrement and Reset buttons visually indicates that they are inactive.

It also has a stepcounter subclass that allows you to increment or decrement the count by certain customized value, demonstrating inheritance and polymorphism.
