# Overview

The **Counter Component** is a reusable JavaScript UI component built using **Object-Oriented Programming (OOP)** principles.  
It demonstrates encapsulation, state-driven UI updates, and DOM manipulation inside a class.

# 🎯 Learning Goals

This assignment focuses on reinforcing:

- Encapsulation
- State-driven UI updates
- DOM manipulation within a class
- Separation of concerns (`mount()` vs `update()`)
- Technical documentation

# ✨ Features

### Base Counter

- Increment button
- Decrement button
- Reset button
- Custom starting value
- Prevents negative numbers
- CSS applied to inactive decrement and reset buttons when it reaches 0
- Multiple counters on the same page

## StepCounter

- Overrides increment() and decrement() methods to use step
- Demonstrates inheritance and polymorphism
